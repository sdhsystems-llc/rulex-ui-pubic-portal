decision tables

  • Import from Excel / CSV — instant conversion
  • Coverage gap analysis — find uncovered input combinations
  • Conditional formatting and cell-level annotations
  • Real-time validation prevents conflicting rules
  • Export to FEEL expressions or JSON schema

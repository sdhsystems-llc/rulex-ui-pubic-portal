Place your pipeline walkthrough video in this folder:

- `rulex-pipeline-walkthrough.mp4`

The home page (`index.html`) uses:

```html
<source src="assets/rulex-pipeline-walkthrough.mp4" type="video/mp4" />
```

Recommended video settings:

- Resolution: 1920x1080 or 1600x900
- Format: MP4 (H.264)
- Duration: 20-60 seconds
- Keep it silent so autoplay works in browsers
